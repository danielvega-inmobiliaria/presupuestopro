# PresupuestoPRO — Instrucciones del proyecto

Al empezar cada chat, leé `PROYECTO.md` para conocer el estado actual del proyecto y continuá desde donde quedamos.

Al terminar cada chat, actualizá `PROYECTO.md` con todo lo que se hizo en la sesión.

Cuando el contador de mensajes llegue a **[Msg 20/20]**, antes de sugerir iniciar un nuevo chat, actualizá automáticamente `PROYECTO.md` con el estado actual del proyecto sin que el usuario tenga que pedirlo.
